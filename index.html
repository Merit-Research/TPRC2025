Data Products and Support Documents for the TPRC2025 paper:

Towards unbiased inference of Internet broadband availability based on observational speed test data

Products

  • Interactive tables: Debiased broadband penetration for all US derived from tract level estiamtes:
    • At the county level: interactive table | data frame (RData, 16.2MB).
    • At the state level: interactive table | data frame (RData, 256KB)
  • Interactive plots: Growth charts
    • At the tract level: Broadband (% connected) | Broadband (% households) | Gini index
    • At the county level: Broadband (% connected) | Broadband (% households) | Gini index
  • Interactive maps: An R Shiny web-app illustrating the debiased broadband inference broadband maps. (Work in progress)
